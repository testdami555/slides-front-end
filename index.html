¿Qué es el desarrollo Front-End?

  • Es la parte del desarrollo web enfocada en la interfaz de usuario.
  • Todo lo que el usuario ve y con lo que interactúa.
  • Objetivo: transformar el diseño en una experiencia funcional.

Front-End vs. Back-End

  • Front-End: apariencia, navegación, interacción.
  • Back-End: lógica del servidor, base de datos, seguridad.
  • Ambos trabajan juntos para formar un sitio web completo.

Sitios Multipágina

  • Sistemas de páginas conectadas entre sí.
  • Ejemplo: sitios de empresas, diarios online, blogs, etc.
  • Ventaja: organizan mucha información.
  • Desventaja: hay que desarrollar más plantillas y puede llevar mayor mantenimiento.

One Page y Landing Pages

  • One Page: toda la información en una sola página con scroll.
  • Landing Page: página con un objetivo puntual (promoción de ventas, servicios o eventos).
  • Muy usadas en marketing digital.

El concepto de Plantillas

  • Diseños ya armados que se pueden reutilizar.
  • Permiten crear sitios más rápido.
  • Base común: estructura en HTML, estilos en CSS, interacción en JS.
  • Ejemplo: plantillas en WordPress.

HTML: la estructura

  • Estrauctura y equeleto del sitio web.
  • Define títulos, párrafos, imágenes, formularios.
  • Sin HTML, la web no tiene contenido visible.

CSS: el estilo

  • Da color, tipografía, diseño y animaciones.
  • Permite que el sitio sea atractivo y adaptable (responsive).
  • Separa el contenido (HTML) del diseño.

JavaScript: la interacción

  • Lenguaje que hace que el sitio sea dinámico.
  • Ejemplos: menús desplegables, sliders, validación de formularios.
  • Base de frameworks modernos.

Herramientas de Desarrollo

  • Editores: Visual Studio Code (más usado).
  • Navegadores: Chrome o alternativas.
  • Control de versiones: Git + GitHub.

La IA en Front-End

  • Generar código más rápido (ej: Gemini, GitHub Copilot).
  • Optimizar diseño y experiencia de usuario.
  • Automatizar pruebas y depuración.
  • El futuro combina creatividad humana con la potencia de la IA.
